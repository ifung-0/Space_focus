# 🌌 Cosmic Focus // Solar System

**Cosmic Focus** is an immersive, interactive 3D solar system explorer combined with a customizable Pomodoro-style productivity timer. Built entirely with HTML, CSS, and Vanilla JavaScript (powered by Three.js), it allows you to stay focused on your tasks while orbiting through the cosmos.

## ✨ Features

* 🕒 **Productivity Timer:** Built-in timer with presets for Focus, Short Break, and Long Break, plus custom duration support and audio alarms.
* 🪐 **Interactive 3D Universe:** Real-time rendering of the Sun, 8 planets, major moons, the Asteroid belt, the Kuiper belt, and a dynamic star field.
* 📊 **Planetary Data:** Click on celestial bodies (or use the bottom selector) to open the Info Panel, displaying real-world stats like distance, orbital period, temperature, gravity, and fun facts.
* 🎮 **Rich Camera & Time Controls:** * Adjust orbital speed and zoom levels.
    * Toggle auto-rotation and pause time completely.
    * Quick-switch between Top, Isometric, and Edge camera views.
* 🎨 **Customization & Theming:** * Choose from multiple color themes: *Dark, Light, Amber, and Cyber*.
    * Apply immersive HUD overlays: *Spaceship, Helmet, and Retro*.
* ⚡ **Performance Settings:** Toggle mesh resolutions (Low, Med, High) to balance visual fidelity with browser performance.
* 📸 **Screenshot Tool:** Capture and download your current view of the solar system with a single click.

## 🚀 Getting Started

You can experience Cosmic Focus directly in your browser without any installation or local servers required. 

**Launch the app here:** 🔗 [**ifung-0.github.io/Space_focus**](https://ifung-0.github.io/Space_focus/)

## 🛠️ Tech Stack

* **Markup / Styling:** HTML5, CSS3 (CSS Variables for dynamic theming)
* **Logic:** Vanilla JavaScript (ES6 Modules)
* **3D Rendering:** [Three.js](https://threejs.org/) (v0.160.0)
* **Fonts:** Segoe UI & Fira Code (via Fontsource CDN)

## ⌨️ Controls & UI Guide

* **Left Click & Drag:** Rotate the camera around the central target.
* **Right Click & Drag:** Pan the camera.
* **Scroll Wheel:** Zoom in and out.
* **Click on a Planet:** Focuses the camera on that planet and brings up its data panel.
* **Bottom Left Menu (⚙️):** Opens the main control panel to adjust the timer, simulation speed, visibility toggles, and themes.
